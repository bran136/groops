Gravityfield2DegreeAmplitudesPlotGrid

This program computes a timeSeries of a time variable gravityfield and saves it as degree amplitudes. The expansion is limited in the range between minDegree and maxDegree inclusivly \begin{equation} \sigma_n = \frac{GM}{R}\left(\frac{R}{r}\right)^{n+1}k_n\sqrt{\sum_{m=0}^n c_{nm}^2+s_{nm}^2}. \end{equation} The outputfileTimeSeries is a matrix with every row containing the time, degree, degree amplitude, and the formal error.

To visualize the results use PlotGraph).

See also Gravityfield2DegreeAmplitudes.
