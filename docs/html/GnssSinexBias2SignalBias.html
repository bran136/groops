GnssSinexBias2SignalBias

Converts GNSS signal biases from IGS SINEX Bias format to GnssSignalBias format.

Only satellite observable-specific signal biases (OSB) are supported at the moment. Time-variable biases (slopes) are ignored. Combined X signals are transformed to actually transmitted signals.

If entries exist for different time periods, you must select one using time.

See also GnssSignalBias2SinexBias.
